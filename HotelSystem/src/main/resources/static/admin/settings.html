规则设置

保证金退还策略

在入住前24小时（含当天）之前取消订单，退还的保证金比例（默认100%，即全额退款）
在入住前24小时内取消订单，退还的保证金比例（默认90%，即扣除10%违约金）
入住当天或之后取消订单，退还的保证金比例（默认0%，即无退款）

节假日折扣策略

节假日期间的折扣率，0表示无折扣，10表示打9折（10%折扣）
节假日日期列表，JSON数组格式，例如：["2024-01-01", "2024-05-01"]
